# Pizza_Business_insights_and_Trends

An end-to-end data analysis project using python to evaluate the performance of a pizza restaurant.
This project involves data cleaning, exploratory data analysis (EDA), and visualisation of sales trends, peak operating hours, and product profitability.

Key Questions Answered:

- What are the peak hours and busiest days for orders?

- Which pizza categories (Classic, Veggie, Supreme, Chicken) contribute most to the bottom line?

- How does average order value (AOV) fluctuate throughout the month?

- What size of pizza is the most frequently purchased across different categories?

Key tools used: 

- Pandas, Matplotlib, numpy, Data Visualization and Seaborn.
